### 启动vi

```
vi
```



### 退出

退出vi，出现未保存文件等情况会无法退出

```
:q
```

强制退出

```
:q!
```

### 保存

```
:w
```

### 插入模式

要在文件中添加文本，我们需要先进入插入模式。按下"i"键进入插入模式。

若要退出插入模式返回命令模式，按下 Esc 按键。
