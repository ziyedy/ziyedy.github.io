格式化输出常用转换符

转 换 符 说 明
%s 字符串类型
%c 字符类型
%d 整数类型（十进制）
%f 浮点类型
%n 或 \n 换行符

标 志 说 明 示 例 结 果

Arrays.sort()——排序

对所有数组元素进行排序（默认正序）

使用lamda表达式更改排序方式，以倒序为例，此时需要注意声明数组必须使用其包装类，否则会报错

对指定索引范围内元素进行排序

Arrays.fill()——填充

为数组元素填充相同值

为指定索引范围内数组填充相同值

Arrays.equals()——判断两个数组各元素是否都相等

Arrays.binarySerch()——二分查找定位元素下标
